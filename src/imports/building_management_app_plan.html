앱 기획서 v1.0 MVP 기준

🏢 빌딩 수익 관리 앱

임대 수익 · 계약 · 납입 · 세금 · 수리 통합 관리 플랫폼

앱 한 줄 정의

건물주가 임대 수익·관리비·세금·수리비를 입력하고, 계약서와 통장 내역을 업로드하면 — 수익 현황 대시보드, 계약 만료 알림, 미납 월세 감지, 6개월 운영 시뮬레이션을 자동으로 제공하는 올인원 자산 관리 앱.

핵심 사용자 흐름

1
건물 등록
주소, 층수, 호실 수, 보증금·임대료 기본 정보 입력
2
자료 업로드
계약서(PDF), 통장 내역(Excel/CSV), 수리·관리비 영수증
3
AI 분석
계약 조건 파싱 → 납입 패턴 매칭 → 수익 자동 계산
4
대시보드 확인
수익 현황, 만료 예정 계약, 미납 세입자, 운영 지침 한눈에
5
알림 & 리포트
계약 만료 D-90/30, 미납 감지, 월간 수익 리포트 자동 발송

타겟 사용자

주 타겟
1~5동 소규모 건물주
부 타겟
부동산 위탁 관리인
관리 유형
상가 · 오피스텔 · 주택
핵심 니즈
세금 신고 · 미납 추적

필수 기능 (MVP)

📊
수익 대시보드
월 임대 수익, 관리비, 세금, 수리비 합산 → 실수익 자동 계산. 전월 대비 증감률 표시.
월별 차트실수익 계산지출 분류
📄
계약서 분석 (AI)
PDF 계약서 업로드 → AI가 임차인명, 보증금, 월세, 계약 기간 자동 추출. 만료일 캘린더 등록.
PDF 파싱만료일 알림자동 등록
🏦
통장 내역 납입 추적
통장 내역 CSV/Excel 업로드 → 세입자별 월세 자동 매칭. 미납·지연 자동 감지 및 알림.
CSV 업로드미납 감지납입 이력
🔔
스마트 알림
계약 만료 D-90, D-30, D-7 알림. 월세 미납 시 자동 안내 문자 템플릿 제공.
푸시 알림문자 템플릿이메일 발송
📅
6개월 운영 시뮬레이션
공실 발생, 수리비 급증, 세금 납부 시기 등 변수 시나리오별 수익 예측 및 대처 지침 제공.
시나리오 분석변수 대처법현금흐름 예측

추가 기능 (고도화)

🧾
세금 신고 자료 자동 생성
종합소득세·부가세 신고용 수입·지출 내역 자동 정리. 세무사 공유용 PDF 리포트 출력.
🔧
수리·유지보수 이력 관리
호실별 수리 내역, 업체 연락처, 보증 기간 기록. 정기 점검 리마인더 설정.
💬
세입자 커뮤니케이션
앱 내 문자 발송, 공지 전달. 세입자별 대화 이력 보관. 민원 접수 및 처리 현황 추적.
📈
수익률 분석 & 부동산 시세
투자 수익률(ROI) 자동 계산. 주변 임대 시세 비교 (공공 데이터 연동). 임대료 조정 가이드.
🤝
다중 사용자 권한
건물주 · 관리인 · 세무사 별도 접근 권한. 세무사에게 특정 기간 자료만 공유 가능.

수익 계산 구조

월 임대 수익 합계 + 5,200,000원
관리비 수입 + 480,000원
관리비 지출 (전기·수도·청소) - 310,000원
수리·유지보수비 - 120,000원
재산세·종합소득세 (월 환산) - 380,000원
대출 이자 (해당 시) - 650,000원
월 실수익 = 4,220,000원

※ 위는 예시 수치입니다. 실제 데이터 입력 시 자동 계산됩니다.

통장 내역 분석 방식

1
파일 업로드
은행 앱에서 내역 다운로드 (CSV·Excel) → 앱 업로드
2
세입자 매칭
입금자명·금액·날짜를 계약서 정보와 자동 매칭. 이름이 다를 경우 수동 확인 요청.
3
미납 감지
매월 납입일 ±3일 이내 미입금 세입자 자동 표시. 지속 미납 시 경고 단계 상향.
4
리포트 생성
세입자별 납입 이력, 미납 횟수, 지연 일수 월간 리포트 자동 생성

미납 단계 알림

1단계 납입일 +3일 초과 문자 알림 템플릿 제공
2단계 납입일 +10일 초과 내용증명 작성 가이드
3단계 연속 2개월 미납 법적 조치 절차 안내

납입 현황 예시

이번 달 납입 완료 8호실
미납 (3일 이내) 1호실
미납 (10일 초과) 1호실
납입율 80%

※ 예시 데이터입니다.

6개월 변수 대처 시뮬레이션

🏚️
공실 발생
공실 기간 수익 자동 차감 → 최소 필요 납입 호실 수 계산. 공실 기간 기준 수리·리모델링 적정 예산 가이드 제공.
🔨
수리비 급증
월 수리비가 임대 수익의 15% 초과 시 경고. 보수 vs 리모델링 손익분기 계산 제공.
📈
금리 인상
대출 이자 변동 입력 시 실수익 재계산. 임대료 조정 필요 금액 자동 안내.
⚖️
세법 변경
임대소득세율 변경 시 영향 시뮬레이션. 절세 전략 (필요경비 최대화) 안내.
🚪
세입자 분쟁
미납·퇴거 거부·원상복구 분쟁 단계별 법적 대응 절차 체크리스트. 내용증명 템플릿 자동 생성.
🌪️
재해·화재
건물 보험 정보 등록 및 보험료 관리. 재해 발생 시 보험 청구 절차 안내.
