---

## 📥 배포 3단계

### 1️⃣ 다운로드

**Figma Make에서:**
1. 좌측 메뉴 → **"Export"** 또는 **"Download"**
2. ZIP 파일 다운로드
3. 압축 해제

### 2️⃣ 배포

**로컬 컴퓨터 터미널에서:**

```bash
# 프로젝트 폴더로 이동
cd /path/to/building-revenue-manager

# 배포 스크립트 실행
# Mac/Linux:
chmod +x deploy.sh && ./deploy.sh

# Windows:
deploy.bat
```

**또는 수동 배포:**

```bash
npm install -g vercel
vercel login
vercel --prod
```

### 3️⃣ 완료!

배포 완료 시 URL이 출력됩니다:
```
https://building-revenue-manager-xxx.vercel.app
```
